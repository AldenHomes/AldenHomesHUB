Setup needed

This tool needs to be connected to a shared Firebase project before your team can use it. Whoever set this up should follow SETUP_GUIDE.md and paste the project config into the firebaseConfig block near the top of this file's script.

1. Create a free project at console.firebase.google.com
2. Add a Web App, copy the config object
3. Enable Firestore Database
4. Enable Authentication → Email/Password
5. Add a user for each teammate
6. Paste the config into this file, replacing the placeholder values
7. Re-upload the file to GitHub
